# レジの助 (reji-no-suke)

モバイルオーダー（客が自身のスマホ等で注文するシステム）のプロトタイプWebアプリケーションです。

## 概要

店舗のテーブルに配置されたQRコードを読み取り（本アプリでは卓番号の入力でシミュレート）、自身の端末からメニューを閲覧・注文できるシステムです。

## 技術スタック

*   **バックエンド**: Node.js, Express
*   **データベース**: SQLite (better-sqlite3)
*   **フロントエンド**: HTML, CSS, Vanilla JavaScript

## 環境構築と起動手順

本アプリは Node.js と SQLite（ファイルベース）で動作します。

### 1. 依存関係のインストール
```bash
npm install
```

### 2. データベースの初期化とシードデータ投入
```bash
npm run seed
```
※ 初期データ（テーブル、メニュー一覧など）が `database.sqlite` に作成されます。データをリセットしたい場合はいつでもこのコマンドを実行してください。

### 3. サーバー起動
```bash
npm start
```
サーバーが `http://localhost:3000` で起動します。

## 利用手順

1. **注文アプリ（顧客画面）**
   *   `http://localhost:3000` または `http://localhost:3000/index.html` にアクセスします。
   *   卓番号（例: `table_1`, `table_2`, `table_3` など）を入力してログインします。
   *   メニュータブから商品を注文し、お会計確認タブで注文履歴を確認できます。

2. **管理画面（店員画面）**
   *   `http://localhost:3000/admin.html` にアクセスします。
   *   全卓の現在の注文状況（合計金額など）を確認できます。
   *   「会計済み」ボタンを押すと、その卓のステータスが更新され、顧客画面側も「退店画面（thankyou.html）」へと自動遷移します。
